Before any syntax, a question: _what can you actually make with NeedleScript?_

The answer is: designs that can't be drawn in traditional embroidery software — organic noise fields, recursive fractals, parametric geometry. And because everything runs on your device, what you preview is _exactly_ what your machine will sew.

## Generative embroidery

Traditional digitizing is drawing. You trace shapes, assign stitch types, tweak by hand. NeedleScript is programming. You describe rules — _how_ a design is built — and the machine resolves them into thread.

This means:

- The same six lines of code can produce twelve completely different pieces by changing a single number (the **seed**).
- You can build things that would take hours to digitize manually — noise-guided flow fields, recursively branching trees, Voronoi-shattered fills — in minutes.
- Every design is reproducible: same code + same seed = identical `.DST` file, every time.
