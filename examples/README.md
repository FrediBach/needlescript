# Playground examples

Each `.ns` file in a tier directory is bundled into the playground automatically.

- Use `intro`, `intermediate`, or `advanced` for the directory name; it determines the picker group.
- The filename (without `.ns`) is the stable example ID.
- Hyphens in filenames become spaces in the generated picker label.
- Examples within a tier are displayed alphabetically by that label.

For example, `examples/intermediate/spiral-study.ns` is selectable as **spiral study**.
